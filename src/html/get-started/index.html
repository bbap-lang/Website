Install

The BBAP compiler is a DotNet tool. To install it, you need to have the latest DotNet SDK installed on your machine.
If you don't have it, you can download it from here.

Once you have the DotNet SDK installed, you can install the BBAP compiler by running the following command:

Using dotnet

dotnet tool install -g bbap

Using Cake

#tool dotnet:?package=bbap

Using Nuke

nuke :add-package bbap

If you don't want to install the compiler, you can try it online.

First Program

Let's write our first program in BBAP. Create a file named hello.bbap and write the following code in it:

let message = "Hello, World!"; println(message);

To compile the program, run the following command:

bbap hello.bbap hello.abap

This will generate a file named hello.abap which is the compiled version of the program. To run the program, you need to copy the content into your SAP system and run it.
